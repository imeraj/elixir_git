# elixir_git
An Elixir implementation of Git version control system

# Installation
  * mix deps.get
  * mix escript.build

# Configuration
  * Set EGIT_AUTHOR_NAME and EGIT_AUTHOR_EMAIL environment variables in your .bashrc or shell config file

# Usage
  * ./egit init [dir]
  * ./egit commit
  * ./egit add file(s)|dir(s)
